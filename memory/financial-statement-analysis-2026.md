## 一、三大报表概览

### 1. 资产负债表（Balance Sheet）

**公式：** 资产 = 负债 + 所有者权益

**核心作用：** 反映公司在某一时点的财务状况（"家底"）

**主要项目：**

| 资产（Assets） | 负债（Liabilities） | 所有者权益（Equity） |
|--------------|-------------------|-------------------|
| 流动资产 | 流动负债 | 实收资本 |
| - 货币资金 | - 短期借款 | - 资本公积 |
| - 应收账款 | - 应付账款 | - 盈余公积 |
| - 存货 | - 预收账款 | - 未分配利润 |
| 非流动资产 | 非流动负债 | |
| - 固定资产 | - 长期借款 | |
| - 无形资产 | - 应付债券 | |
| - 商誉 | | |

**关键分析点：**
- 资产负债率 = 负债/资产（警戒线：70%）
- 流动比率 = 流动资产/流动负债（健康：>2）
- 速动比率 = (流动资产 - 存货)/流动负债（健康：>1）

---

### 2. 利润表（Income Statement）

**公式：** 净利润 = 收入 - 成本 - 费用 - 税金

**核心作用：** 反映公司在一定时期的经营成果（"赚钱能力"）

**主要项目：**

```
营业收入
├── 营业成本
├── 税金及附加
├── 销售费用
├── 管理费用
├── 研发费用
├── 财务费用
├── 投资收益
└── 营业外收支
    ↓
营业利润
├── 营业外收入
└── 营业外支出
    ↓
利润总额
├── 所得税费用
    ↓
净利润
├── 少数股东损益
    ↓
归母净利润
```

**关键分析点：**
- 毛利率 = (收入 - 成本) / 收入
- 净利率 = 净利润 / 收入
- ROE = 净利润 / 净资产
- 营收增速、净利润增速

---

### 3. 现金流量表（Cash Flow Statement）

**核心作用：** 反映公司在一定时期的现金流入流出情况（"真金白银"）

**三大活动现金流：**

| 活动类型 | 现金流入 | 现金流出 | 健康状态 |
|----------|----------|----------|----------|
| 经营活动 | 销售商品、提供劳务 | 采购、工资、税费 | 应为正 |
| 投资活动 | 收回投资、处置资产 | 购建资产、对外投资 | 可负（扩张期） |
| 筹资活动 | 借款、发行股票 | 还债、分红、回购 | 可正可负 |

**关键分析点：**
- 经营现金流净额 > 净利润（利润质量高）
- 自由现金流 = 经营现金流 - 资本支出
- 现金及等价物净增加额 = 三大活动现金流之和

## 二、财务比率分析

### 1. 偿债能力指标

| 指标 | 公式 | 健康值 | 含义 |
|------|------|--------|------|
| 流动比率 | 流动资产/流动负债 | >2 | 短期偿债能力 |
| 速动比率 | (流动资产 - 存货)/流动负债 | >1 | 快速偿债能力 |
| 资产负债率 | 负债/资产 | <70% | 长期偿债能力 |
| 利息保障倍数 | 息税前利润/利息费用 | >3 | 支付利息能力 |
| 现金比率 | 货币资金/流动负债 | >0.5 | 最保守偿债能力 |

**风险信号：**
- 流动比率 < 1：短期偿债压力大
- 资产负债率 > 70%：财务风险高
- 利息保障倍数 < 1：可能无法支付利息

---

### 2. 盈利能力指标

| 指标 | 公式 | 优秀值 | 含义 |
|------|------|--------|------|
| 毛利率 | (收入 - 成本)/收入 | >30% | 产品竞争力 |
| 净利率 | 净利润/收入 | >15% | 整体盈利能力 |
| ROE | 净利润/净资产 | >15% | 股东回报率 |
| ROA | 净利润/总资产 | >8% | 资产使用效率 |
| ROIC | 息税后利润/投入资本 | >12% | 资本使用效率 |

**杜邦分析：**
```
ROE = 净利率 × 资产周转率 × 权益乘数
    = (净利润/收入) × (收入/资产) × (资产/净资产)
```

**风险信号：**
- 毛利率持续下降：竞争加剧或成本上升
- ROE < 10%：股东回报偏低
- 净利率 < 5%：盈利能力弱

---

### 3. 运营效率指标

| 指标 | 公式 | 含义 |
|------|------|------|
| 应收账款周转率 | 收入/平均应收账款 | 收款速度 |
| 存货周转率 | 成本/平均存货 | 存货销售速度 |
| 总资产周转率 | 收入/平均总资产 | 资产使用效率 |
| 固定资产周转率 | 收入/平均固定资产 | 固定资产使用效率 |

**风险信号：**
- 应收账款周转率下降：回款变慢，可能有坏账
- 存货周转率下降：产品滞销，可能减值

---

### 4. 估值指标

| 指标 | 公式 | 适用场景 | 合理范围 |
|------|------|----------|----------|
| PE（市盈率） | 股价/每股收益 | 盈利稳定公司 | 10-30 倍 |
| PB（市净率） | 股价/每股净资产 | 重资产公司 | 1-5 倍 |
| PS（市销率） | 市值/营业收入 | 未盈利成长公司 | 5-20 倍 |
| PEG | PE/净利润增速 | 成长股 | 0.5-2 |
| EV/EBITDA | 企业价值/息税折旧摊销前利润 | 重资产/高负债 | 5-15 倍 |

## 三、财务造假识别

### 常见造假手法

#### 1. 虚增收入

**手法：**
- 提前确认收入（发货前确认）
- 虚构客户和交易
- 关联方交易非关联化
- 一次确认多次收入

**预警信号：**
- 应收账款增速 >> 收入增速
- 经营现金流净额 << 净利润
- 客户集中度高且新增大客户
- 毛利率远高于同行

---

#### 2. 虚减成本

**手法：**
- 推迟确认成本
- 少结转存货成本
- 资本化本应费用化的支出
- 少提折旧摊销

**预警信号：**
- 存货增速 >> 成本增速
- 固定资产/在建工程异常增加
- 毛利率异常高于同行
- 折旧率低于同行

---

#### 3. 虚增资产

**手法：**
- 虚构货币资金
- 高估存货价值
- 无形资产/商誉不减值
- 应收账款不计提坏账

**预警信号：**
- 货币资金高但利息收入低（康美药业案）
- 存货周转率持续下降
- 商誉占净资产比例高
- 应收账款账龄老化

---

#### 4. 隐瞒负债

**手法：**
- 表外负债（担保、明股实债）
- 推迟确认应付账款
- 少计预计负债

**预警信号：**
- 对外担保金额大
- 其他应付款异常增加
- 少数股东权益异常
- 表外融资迹象

---

### 财务造假预警信号汇总

**🚨 高危信号（出现 3 个以上需高度警惕）：**

1. **利润与现金流背离** - 连续多年净利润为正但经营现金流为负
2. **应收账款异常** - 应收账款增速远超收入增速
3. **存货异常** - 存货增速远超成本增速
4. **毛利率异常** - 远高于同行且无合理解释
5. **货币资金悖论** - 账上现金多但利息收入低
6. **频繁更换审计机构** - 尤其是换到小所
7. **高管频繁离职** - 尤其是 CFO/财务总监
8. **大股东高比例质押** - 质押率 > 70%
9. **关联交易占比高** - 收入/采购依赖关联方
10. **商誉占比高** - 商誉/净资产 > 30%

## 四、财务报表分析框架

### 五步分析法

```
第一步：看规模
├── 总资产、净资产规模
├── 营业收入规模
└── 净利润规模

第二步：看增长
├── 营收增速（3 年/5 年 CAGR）
├── 净利润增速
└── 经营现金流增速

第三步：看质量
├── 毛利率、净利率
├── ROE、ROA
└── 经营现金流/净利润

第四步：看风险
├── 资产负债率
├── 流动比率、速动比率
└── 有息负债率

第五步：看估值
├── PE、PB、PS
├── PEG
└── 与同行对比
```

## 五、案例分析

### 康美药业财务造假案（2019 年曝光）

**造假手法：**
- 虚增货币资金 887 亿元
- 虚增营业收入 291 亿元
- 虚增利润 39 亿元

**预警信号（事后看）：**
- 货币资金 340 亿，但利息收入仅 1000 万（年化 0.03%）
- 应收账款增速远超收入增速
- 存货异常增加
- 大股东高比例质押

**教训：** 货币资金高但利息收入低是典型造假信号

---

### 贵州茅台（优质财务典范）

**财务特征：**
- 毛利率 > 90%（行业最高）
- 净利率 > 50%
- ROE > 30%
- 经营现金流净额 > 净利润
- 资产负债率 < 30%
- 货币资金充裕，无有息负债

**启示：** 优质公司的财务报表是"三高一无"——高毛利、高净利、高 ROE、无有息负债
