## 📈 二、观察指标体系

### 2.1 每日检查（5 分钟）

**价格/成交量：**
- [ ] 收盘价是否在关键均线上方（20 日/50 日/200 日）
- [ ] 成交量是否异常放大（>5 日均量 2 倍）
- [ ] 是否有跳空缺口（向上/向下）

**新闻/事件：**
- [ ] 公司是否有重大公告（财报/并购/诉讼）
- [ ] 行业是否有政策变化
- [ ] 宏观是否有重要数据发布（CPI/非农/美联储）

**快速决策矩阵：**
| 情况 | 操作 |
|------|------|
| 股价跌破 200 日均线 | ⚠️ 警惕，检查基本面 |
| 成交量异常放大 + 利空 | 🔴 考虑减仓 |
| 成交量异常放大 + 利好 | 🟢 可能是突破信号 |
| 无明显变化 | ✅ 持有观望 |

### 2.2 每周检查（30 分钟）

**技术面：**
- [ ] 周 K 线趋势判断（上涨/下跌/震荡）
- [ ] 关键支撑/阻力位更新
- [ ] MACD/RSI 指标状态

**基本面：**
- [ ] 本周是否有财报发布
- [ ] 分析师评级变化
- [ ] 机构持仓变化（13F 文件）

**行业/宏观：**
- [ ] 行业指数表现对比
- [ ] 竞争对手动态
- [ ] 宏观经济数据

### 2.3 每月检查（2 小时）

**组合复盘：**
- [ ] 本月收益率计算
- [ ] 与基准对比（沪深 300/纳指 100）
- [ ] 仓位检查（是否偏离目标配置）

**基本面深度检查：**
- [ ] 财报关键指标更新（营收/利润/毛利率）
- [ ] 估值水平更新（PE/PEG/PB）
- [ ] 竞争优势是否变化

**风险管理：**
- [ ] 单只股票仓位是否>20%
- [ ] 行业集中度是否>40%
- [ ] 止损位是否需要调整

### 2.4 每季度检查（半天）

**深度复盘：**
- [ ] 季度收益率分析
- [ ] 成功/失败交易案例总结
- [ ] 投资策略是否需要调整

**基本面重估：**
- [ ] 公司竞争优势是否持续
- [ ] 行业景气度变化
- [ ] 宏观经济周期判断

**策略调整：**
- [ ] 是否需要调仓换股
- [ ] 是否需要调整仓位
- [ ] 是否需要调整止损/止盈位

- ❌ **实时交易** — 无法执行买卖操作
- ❌ **持牌建议** — 不是持牌投资顾问

## 🎯 五、实战建议（新手起步）

### 5.1 第一阶段（1-3 个月）：学习 + 模拟

**目标：** 熟悉市场，建立纪律

**行动：**
1. 开立模拟账户（同花顺/东方财富都有）
2. 每日执行"每日检查"清单
3. 记录每笔"交易"的理由和结果
4. 每周复盘，总结成功/失败

**预期收获：**
- 熟悉交易软件操作
- 建立检查清单习惯
- 理解市场波动

### 5.2 第二阶段（3-6 个月）：小仓位实战

**目标：** 验证策略，控制风险

**行动：**
1. 用小仓位（总资金 10-20%）开始实战
2. 严格执行止损（-8% 强制卖出）
3. 每日/每周检查清单
4. 每月深度复盘

**预期收获：**
- 体验真实市场情绪
- 验证投资框架有效性
- 建立风险控制意识

### 5.3 第三阶段（6 个月+）：逐步加仓

**目标：** 稳定盈利，优化策略

**行动：**
1. 根据表现逐步加仓（每次 10-20%）
2. 优化选股标准和买卖时机
3. 建立个人投资体系
4. 每季度深度复盘

**预期收获：**
- 形成稳定投资风格
- 实现长期正收益
- 建立投资自信
