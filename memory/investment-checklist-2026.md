## 一、选股检查清单（买入前必查）

### 基本面检查（70% 权重）

#### 公司质地
- [ ] **行业地位：** 是否为行业龙头或细分冠军？
- [ ] **竞争优势：** 是否有护城河（品牌/技术/规模/网络效应）？
- [ ] **商业模式：** 是否简单易懂？盈利模式是否清晰？
- [ ] **管理层：** 是否诚信？过往记录如何？

#### 财务健康
- [ ] **营收增长：** 过去 3 年营收是否持续增长？（>10% 为佳）
- [ ] **利润增长：** 过去 3 年净利润是否持续增长？（>15% 为佳）
- [ ] **毛利率：** 是否稳定或提升？（>30% 为佳）
- [ ] **ROE：** 净资产收益率是否>15%？
- [ ] **负债率：** 资产负债率是否<60%？（金融股除外）
- [ ] **现金流：** 经营现金流是否为正？是否>净利润？
- [ ] **应收账款：** 是否增长过快？（警惕财务造假）

#### 估值水平
- [ ] **PE（市盈率）：** 是否处于历史低位？（<历史 50% 分位）
- [ ] **PB（市净率）：** 是否合理？（<3 为佳，周期股除外）
- [ ] **PEG：** 是否<1？（成长股适用）
- [ ] **股息率：** 是否有分红？（>2% 为佳，非必需）

### 技术面检查（30% 权重）

#### 趋势判断
- [ ] **大趋势：** 周线/月线是否向上？（顺大势）
- [ ] **均线排列：** 是否多头排列？（短期>中期>长期）
- [ ] **关键位置：** 是否在支撑位附近？（非高位追涨）

#### 买卖信号
- [ ] **K 线形态：** 是否有底部反转信号？（锤子线/吞没/晨星）
- [ ] **指标共振：** 是否至少 2 个指标发出买入信号？（MACD/RSI/KDJ）
- [ ] **成交量：** 是否放量上涨/缩量回调？
- [ ] **突破确认：** 如有突破，是否放量确认？

### 风险排查（一票否决）

#### 红色警报（出现即放弃）
- [ ] **财务造假嫌疑：** 应收账款异常/存货异常/现金流为负
- [ ] **大股东高比例质押：** 质押率>50%
- [ ] **频繁减持：** 大股东/高管近期频繁减持
- [ ] **监管处罚：** 近期被证监会处罚或立案调查
- [ ] **业绩暴雷：** 近期业绩大幅低于预期
- [ ] **商誉过高：** 商誉/净资产>30%
- [ ] **诉讼风险：** 重大未决诉讼

#### 黄色警报（需谨慎）
- [ ] **解禁压力：** 近期有大额限售股解禁
- [ ] **行业政策风险：** 行业面临政策调整
- [ ] **客户集中度高：** 单一客户占比>50%
- [ ] **周期性下行：** 行业处于周期下行阶段

## 二、买入检查清单（下单前必查）

### 买入理由
- [ ] **明确理由：** 能否用一句话说清为什么买？
- [ ] **预期收益：** 预期收益率是多少？（如：30%）
- [ ] **目标价位：** 目标价是多少？依据是什么？
- [ ] **时间周期：** 预计持有多久？（短线/中线/长线）

### 仓位管理
- [ ] **总仓位：** 该股票占总仓位比例？（单只<20%）
- [ ] **分批建仓：** 是否分批买入？（至少 2-3 批）
- [ ] **首批仓位：** 首批买入比例？（建议 30-50%）
- [ ] **加仓计划：** 什么情况下加仓？加仓比例？

### 止损设置
- [ ] **止损位：** 止损价是多少？（技术位/百分比）
- [ ] **止损幅度：** 最大可接受亏损？（建议<10%）
- [ ] **止损纪律：** 能否严格执行止损？

### 买入执行
- [ ] **买入时间：** 什么时间买入？（开盘/收盘/盘中）
- [ ] **买入价格：** 限价单还是市价单？
- [ ] **检查大盘：** 大盘是否稳定？（避免系统性风险）
- [ ] **检查情绪：** 市场情绪是否极端？（避免追高杀跌）

## 三、持仓检查清单（定期复盘）

### 每日检查（短线交易者）
- [ ] **股价走势：** 是否符合预期？
- [ ] **成交量：** 是否异常放量/缩量？
- [ ] **消息面：** 是否有重大消息？
- [ ] **止损位：** 是否触及止损？

### 每周检查（中线交易者）
- [ ] **周线趋势：** 趋势是否改变？
- [ ] **技术指标：** 是否出现背离/死叉？
- [ ] **行业动态：** 行业是否有变化？
- [ ] **仓位调整：** 是否需要加减仓？

### 每月检查（长线投资者）
- [ ] **基本面：** 公司基本面是否恶化？
- [ ] **财报检查：** 最新财报是否符合预期？
- [ ] **估值水平：** 估值是否过高？
- [ ] **投资逻辑：** 买入逻辑是否还在？

### 季度检查（所有投资者）
- [ ] **财报分析：** 季报/年报深度分析
- [ ] **行业地位：** 竞争格局是否变化？
- [ ] **管理层变动：** 核心管理层是否稳定？
- [ ] **组合复盘：** 整体组合表现如何？

## 四、卖出检查清单（决策前必查）

### 止盈卖出
- [ ] **达到目标价：** 是否达到预期目标价？
- [ ] **估值过高：** PE/PB 是否远超历史均值？
- [ ] **技术见顶：** 是否出现顶部信号？（射击之星/吞没/暮星）
- [ ] **指标背离：** MACD/RSI 是否顶背离？
- [ ] **情绪极端：** 市场情绪是否过度乐观？

### 止损卖出
- [ ] **触及止损位：** 是否跌破止损价？
- [ ] **买入逻辑破坏：** 买入理由是否不再成立？
- [ ] **基本面恶化：** 业绩/财务是否恶化？
- [ ] **行业变差：** 行业是否面临重大利空？

### 调仓卖出
- [ ] **发现更好机会：** 是否有更高性价比的标的？
- [ ] **仓位再平衡：** 是否需要降低单一持仓比例？
- [ ] **资金需求：** 是否需要现金？

### 卖出执行
- [ ] **分批卖出：** 是否分批卖出？（避免一次性卖在低点）
- [ ] **卖出时间：** 什么时间卖出？（开盘/收盘/盘中）
- [ ] **税务考虑：** 是否考虑印花税/资本利得税？

---

## 五、风险管理检查清单

### 组合层面
- [ ] **总仓位：** 股票总仓位是否合理？（根据市场环境调整）
- [ ] **行业分散：** 是否过度集中单一行业？（单行业<40%）
- [ ] **个股分散：** 是否过度集中单一个股？（单只<20%）
- [ ] **相关性：** 持仓股票相关性是否过高？

### 风险指标
- [ ] **最大回撤：** 组合最大回撤是否可接受？（<20% 为佳）
- [ ] **波动率：** 组合波动是否过大？
- [ ] **夏普比率：** 风险调整后收益是否合理？（>1 为佳）

### 风险控制
- [ ] **止损纪律：** 是否能严格执行止损？
- [ ] **不加杠杆：** 是否使用杠杆？（建议不用）
- [ ] **不借钱投资：** 是否用闲钱投资？
- [ ] **情绪管理：** 是否被贪婪/恐惧控制？

## 七、哈基偷的投资原则清单

### 核心原则（必须遵守）
- [ ] **不懂不投：** 不理解的公司不买
- [ ] **闲钱投资：** 不用急用的钱投资
- [ ] **分散投资：** 不把所有鸡蛋放一个篮子
- [ ] **长期主义：** 不做短线投机
- [ ] **风险控制：** 本金安全第一，收益第二

### 行为准则（建议遵守）
- [ ] **不追涨杀跌：** 避免情绪化交易
- [ ] **不频繁交易：** 减少交易成本
- [ ] **不借钱投资：** 不用杠杆
- [ ] **不盲从他人：** 独立思考决策
- [ ] **持续学习：** 不断更新知识

### 心态管理
- [ ] **接受亏损：** 亏损是投资的一部分
- [ ] **不贪婪：** 达到目标就止盈
- [ ] **不恐惧：** 优质公司下跌是机会
- [ ] **有耐心：** 好投资需要时间
- [ ] **平常心：** 不因涨跌影响情绪
