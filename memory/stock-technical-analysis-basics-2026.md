## 一、技术分析三大假设

### 假设 1：市场行为包容一切
- 所有信息（经济/政治/心理）都已反映在价格中
- 无需研究原因，只需研究结果（价格走势）
- 与基本面分析的根本分歧点

### 假设 2：价格沿趋势移动
- 价格变动不是随机的，是有趋势的
- 趋势一旦形成，倾向于持续
- 技术分析的核心就是识别和跟随趋势

### 假设 3：历史会重演
- 市场参与者心理模式相对稳定
- 相似的市场情境会产生相似的价格形态
- 这是形态分析的理论基础

## 二、K 线基础

### 单根 K 线构成
```
    上影线（最高价 - 开盘/收盘价）
       │
   ┌───┴───┐  实体（开盘价与收盘价之间）
   │       │
   └───┬───┘
       │
    下影线（最低价 - 开盘/收盘价）
```

- **阳线（红色）：** 收盘价 > 开盘价
- **阴线（绿色）：** 收盘价 < 开盘价

### 核心单 K 线形态

| 形态 | 特征 | 信号 |
|------|------|------|
| 锤子线 | 长下影 + 小实体 + 短上影 | 底部反转（看涨） |
| 射击之星 | 长上影 + 小实体 + 短下影 | 顶部反转（看跌） |
| 十字星 | 开盘≈收盘，上下影线 | 多空平衡，变盘信号 |
| 大阳线 | 长实体，几乎无影线 | 强势上涨 |
| 大阴线 | 长实体，几乎无影线 | 强势下跌 |

### 核心 K 线组合

| 组合 | 构成 | 信号 |
|------|------|------|
| 吞没形态 | 第二根 K 线完全吞没第一根 | 强烈反转信号 |
| 晨星 | 大阴 + 小星 + 大阳 | 底部反转（看涨） |
| 暮星 | 大阳 + 小星 + 大阴 | 顶部反转（看跌） |
| 红三兵 | 连续三根阳线，逐步走高 | 强势上涨延续 |
| 三只乌鸦 | 连续三根阴线，逐步走低 | 强势下跌延续 |

## 三、均线系统（MA）

### 常用均线周期
- **短期：** 5 日（周线）、10 日（半月线）、20 日（月线）
- **中期：** 60 日（季线）、120 日（半年线）
- **长期：** 250 日（年线）

### 均线核心用法

**1. 支撑与压力**
- 股价在均线上方 → 均线是支撑
- 股价在均线下方 → 均线是压力

**2. 金叉与死叉**
- **金叉：** 短周期均线上穿长周期均线（看涨）
- **死叉：** 短周期均线下穿长周期均线（看跌）

**3. 多头排列**
- 短期 > 中期 > 长期均线，向上发散
- 最强上涨趋势信号

**4. 空头排列**
- 短期 < 中期 < 长期均线，向下发散
- 最强下跌趋势信号

### 格兰维尔八大法则
1. 均线从下降走平，股价上穿均线 → 买入
2. 股价跌破均线但均线仍向上 → 观察
3. 股价回调至均线上方获支撑 → 买入
4. 股价暴跌远离均线 → 反弹买入
5. 均线从上升走平，股价跌破均线 → 卖出
6. 股价上穿均线但均线仍向下 → 观察
7. 股价反弹至均线下方遇压力 → 卖出
8. 股价暴涨远离均线 → 回调卖出

## 四、成交量分析

### 量价关系核心原则

| 价格 | 成交量 | 含义 |
|------|--------|------|
| 上涨 | 放量 | 健康上涨，趋势延续 |
| 上涨 | 缩量 | 上涨乏力，可能见顶 |
| 下跌 | 放量 | 恐慌抛售，可能见底 |
| 下跌 | 缩量 | 下跌延续，无人接盘 |
| 横盘 | 缩量 | 观望情绪，等待方向 |
| 横盘 | 放量 | 变盘前兆，选择方向 |

### 成交量核心形态
- **放量突破：** 价格突破关键位置 + 成交量放大 = 有效突破
- **量价背离：** 价格创新高但成交量萎缩 = 趋势可能反转
- **天量天价：** 历史最大成交量对应历史最高价 = 顶部信号
- **地量地价：** 历史最小成交量对应历史最低价 = 底部信号

## 五、核心技术指标

### MACD（平滑异同移动平均线）

**构成：**
- **DIF（快线）：** 12 日 EMA - 26 日 EMA
- **DEA（慢线）：** DIF 的 9 日 EMA
- **MACD 柱：** (DIF - DEA) × 2

**用法：**
- DIF 上穿 DEA → 金叉（看涨）
- DIF 下穿 DEA → 死叉（看跌）
- **顶背离：** 价格创新高，MACD 不创新高 → 看跌
- **底背离：** 价格创新低，MACD 不创新低 → 看涨

### RSI（相对强弱指标）

**范围：** 0-100

**用法：**
- RSI > 70 → 超买区，可能回调
- RSI < 30 → 超卖区，可能反弹
- RSI = 50 → 多空平衡
- **背离信号：** 与 MACD 类似，可靠性高

### KDJ（随机指标）

**构成：** K 线（快）、D 线（慢）、J 线（超快）

**用法：**
- K/D > 80 → 超买
- K/D < 20 → 超卖
- J > 100 或 < 0 → 极端信号
- 金叉/死叉信号频繁，需配合其他指标

### BOLL（布林带）

**构成：**
- **中轨：** 20 日均线
- **上轨：** 中轨 + 2 倍标准差
- **下轨：** 中轨 - 2 倍标准差

**用法：**
- 股价触及上轨 → 可能回调
- 股价触及下轨 → 可能反弹
- **收口：** 波动率降低，变盘前兆
- **开口：** 波动率放大，趋势延续

## 六、趋势线与通道

### 趋势线画法
- **上升趋势线：** 连接两个或以上低点，向右上方延伸
- **下降趋势线：** 连接两个或以上高点，向右下方延伸

### 趋势线有效性
- 触点越多越有效（至少 2 点，3 点以上最佳）
- 角度适中（约 45 度最健康）
- 被测试次数越多，突破后信号越强

### 通道理论
- **上升通道：** 趋势线 + 平行上轨，买入机会在下轨
- **下降通道：** 趋势线 + 平行下轨，卖出机会在上轨
- **水平通道：** 箱体震荡，高抛低吸

## 七、形态分析

### 反转形态

| 形态 | 特征 | 信号 |
|------|------|------|
| 头肩顶 | 左肩 + 头 + 右肩，跌破颈线 | 顶部反转（看跌） |
| 头肩底 | 左肩 + 头 + 右肩，突破颈线 | 底部反转（看涨） |
| 双顶（M 头） | 两个相近高点，跌破颈线 | 顶部反转 |
| 双底（W 底） | 两个相近低点，突破颈线 | 底部反转 |
| 圆弧顶 | 缓慢形成弧形顶部 | 顶部反转 |
| 圆弧底 | 缓慢形成弧形底部 | 底部反转 |

### 中继形态

| 形态 | 特征 | 信号 |
|------|------|------|
| 三角形 | 收敛震荡，选择方向突破 | 趋势延续 |
| 矩形 | 水平箱体震荡 | 突破后延续原趋势 |
| 旗形 | 陡峭趋势后的反向小通道 | 趋势延续 |
| 楔形 | 收敛的倾斜通道 | 通常与原趋势相反 |

## 八、技术分析使用原则

### ✅ 正确用法
1. **多重确认：** 不依赖单一指标，多指标共振
2. **趋势为王：** 顺大势，逆小势
3. **量价配合：** 价格信号需成交量确认
4. **止损纪律：** 技术破位必须止损
5. **周期共振：** 多周期（日/周/月）同向信号更可靠

### ❌ 常见误区
1. **指标堆砌：** 用太多指标，互相矛盾
2. **忽视大势：** 在下跌趋势中找买入信号
3. **过度交易：** 频繁进出，忽略趋势
4. **忽视基本面：** 纯技术面忽略公司质地
5. **追求完美：** 想抓住每一个波动

## 九、哈基偷的技术分析框架

```
技术分析决策流程
├── 1. 确定大趋势（周线/月线）
├── 2. 识别关键位置（支撑/压力）
├── 3. 等待信号（K 线形态 + 指标共振）
├── 4. 成交量确认
├── 5. 设定止损位
└── 6. 执行交易计划
```

### 哈基偷的技术分析检查清单
- [ ] 大趋势方向明确（多周期一致）
- [ ] 关键支撑/压力位清晰
- [ ] K 线形态发出信号
- [ ] 至少 2 个指标共振（如 MACD+RSI）
- [ ] 成交量配合（突破放量/回调缩量）
- [ ] 止损位明确（技术破位点）
- [ ] 风险收益比合理（至少 1:2）

## 十、与基本面分析的结合

### 哈基偷的完整分析框架
```
投资决策 = 基本面（70%）+ 技术面（30%）

基本面决定：买什么（选股）
技术面决定：何时买（择时）

优质公司 + 好价格 + 好时机 = 完美投资
```

### 使用场景
- **长线投资：** 基本面为主，技术面辅助择时
- **中线波段：** 基本面 + 技术面并重
- **短线交易：** 技术面为主，基本面排雷
