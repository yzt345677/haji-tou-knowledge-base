## 🎨 设计风格识别（67 种 UI 风格）

### 主风格：**Gradient + Modern Tech + Data-Driven**

| 风格 | 应用区域 | 特点 |
|------|----------|------|
| **Gradient** | Hero 区/背景 | 流动的彩色渐变（紫/蓝/粉） |
| **Modern Tech** | 整体视觉 | 现代科技感，线条简洁 |
| **Data-Driven** | 数据展示 | 大数字 + 图表可视化 |
| **Card-Based** | 功能模块 | 卡片式布局，信息分组 |
| **Minimalism** | 文字排版 | 极简文字，聚焦核心信息 |

### 配色方案（Tech Gradient 主题）

**主色调：**
- 背景：白色 `#FFFFFF` / 浅色 `#F6F9FC`
- 强调色：紫色 `#635BFF` / 蓝色 `#00D4FF` / 粉色 `#FF76B8`
- 文字：深色 `#0A2540`（主标题）/ 灰色 `#425466`（次要文字）
- 渐变：`linear-gradient(130deg, #00D4FF, #635BFF, #FF76B8)`

**对比度检查：** ✅ 符合 WCAG AA 标准（深色文字 on 浅色背景）

## 🎯 核心功能 UI 分析

### 1. 支付结账界面（Checkout）

**设计亮点：**
- **多支付方式展示：** 银行卡/Klarna/Affirm/Cash App/加密货币/PayPay/便利店
- **多语言支持：** 中文/英文/德文/日文自动切换
- **订单摘要清晰：** 小计/税费/运费/总计分层展示
- **移动端优化：** 大按钮，易点击

**配色：**
- 主按钮：紫色 `#635BFF`
- 次要按钮：白色背景 + 灰色边框
- 成功状态：绿色 `#32D583`

### 2. 按用量计费界面（Usage-Based Billing）

**设计特点：**
- **用量可视化：** 柱状图显示过去 30 天用量
- **价格透明：** US$0.01 / 1,000 单位
- **套餐对比：** Pro 计划 vs 其他计划
- **实时计算：** 用量变化即时反映费用

**图表设计：**
- 柱状图：紫色渐变
- 坐标轴：浅灰色
- 数据标签：深色清晰

## 💡 哈基偷的学习心得

### Stripe 做得好的地方

1. **渐变色彩的极致运用**
   - 不是单一紫色，是流动渐变（紫→蓝→粉）
   - 渐变用于背景/按钮/图表，形成品牌识别

2. **数据可视化能力**
   - 大数字 + 小说明，层次清晰
   - 柱状图/折线图简洁专业
   - 实时数据展示（API 请求数/交易量）

3. **多语言/多货币支持**
   - 自动检测用户地区
   - 结账界面多语言切换
   - 货币格式本地化（US$/€/¥）

4. **开发者体验优先**
   - 文档清晰，代码示例丰富
   - 三种集成路径（无代码/预集成/自建）
   - GitHub/SDK/API 完整生态

5. **客户信任建立**
   - 客户 Logo 轮播（OpenAI/Amazon/Nvidia）
   - 数据透明（1.9 万亿/99.999%）
   - 案例详细（Hertz/URBN/Instacart）

### 与 Linear 的对比

| 维度 | Linear | Stripe |
|------|--------|--------|
| **主色调** | 深色模式（黑/灰/蓝） | 浅色模式（白/紫/渐变） |
| **目标用户** | 开发者/产品团队 | 企业/开发者/初创公司 |
| **设计风格** | Minimalism + Dark Mode | Gradient + Modern Tech |
| **信息密度** | 高密度（工具型） | 中密度（营销型） |
| **CTA 策略** | 单 CTA（Get started） | 双 CTA（立即开始/联系销售） |
| **数据展示** | 内部数据（Issue 数/Cycle time） | 公开数据（交易额/API 请求） |

## 📋 避坑指南（Stripe 避免的设计）

### ❌ 不做的东西

1. **无复杂导航**
   - 顶部导航不超过 5 项
   - 无汉堡菜单（桌面端）
   - 无多级下拉

2. **无信息过载**
   - 每屏聚焦一个核心信息
   - 使用留白分隔内容
   - 渐进式披露（点击展开详情）

3. **无低对比度文字**
   - 文字/背景对比度 > 4.5:1
   - 无浅灰色文字 on 白色背景
   - 关键信息用深色

4. **无装饰性动画**
   - 只有功能性动画（hover/加载）
   - 无自动播放视频/轮播
   - 支持 prefers-reduced-motion
