## 🧠 量子神经网络（QNN）

### 核心概念

**量子神经网络（Quantum Neural Network, QNN）** 是量子计算原理与神经网络架构的融合产物。

**与传统神经网络的对比：**

| 特性 | 经典神经网络 | 量子神经网络 |
|------|-------------|-------------|
| **基本单元** | 神经元（激活函数） | 量子比特（量子门） |
| **状态表示** | 实数向量 | 量子态（复数向量） |
| **并行性** | 有限并行 | 量子并行（叠加态） |
| **表达能力** | 受限于网络深度 | 高维希尔伯特空间 |
| **训练方法** | 反向传播 | 参数化量子电路 + 经典优化 |

### QNN 工作原理

**基本结构：**
1. **编码层** — 将经典数据编码为量子态
2. **变分层** — 参数化量子门（可训练参数）
3. **测量层** — 测量量子态得到输出

**数学表达：**
```
|ψ(x)⟩ = U(θ)|x⟩
输出 = ⟨ψ(x)|M|ψ(x)⟩
```
其中 U(θ) 是参数化量子电路，M 是测量算符

### 优势与应用

| 优势 | 说明 |
|------|------|
| **高维特征空间** | 量子态天然映射到高维空间，类似"量子核方法" |
| **纠缠捕捉关联** | 量子纠缠可捕捉变量间复杂关联 |
| **参数效率高** | 少量量子比特可表示大量参数 |

**应用场景：**
- 图像识别（量子卷积神经网络）
- 自然语言处理（量子 Transformer）
- 药物分子生成（量子生成对抗网络 QGAN）
- 金融时间序列预测

## 🔧 TensorFlow Quantum（TFQ）

### 什么是 TFQ？

**TensorFlow Quantum** 是 Google 2020 年 3 月推出的开源库，旨在将量子计算与机器学习结合。

**核心目标：**
- 为开发者提供量子机器学习的工具
- 支持混合量子 - 经典神经网络
- 帮助发现 NISQ（含噪声中等规模量子）时代的算法

### TFQ 架构

**三层架构：**
1. **Cirq** — 量子电路设计（Google 的量子计算框架）
2. **TensorFlow** — 经典机器学习后端
3. **TFQ 层** — 连接量子与经典的桥梁

**工作流程：**
```
经典数据 → 编码为量子态 → 量子电路处理 → 测量 → 经典输出 → 损失计算 → 反向传播
```

### 核心功能

| 功能 | 说明 |
|------|------|
| **量子数据表示** | `tfq.convert_to_tensor()` 将量子电路转为张量 |
| **量子层** | `tfq.layers.Expectation` 计算期望值 |
| **量子采样** | `tfq.layers.Sample` 从量子电路采样 |
| **混合模型** | 量子层与经典 Keras 层无缝集成 |

### 代码示例（简化版）

```python
import tensorflow_quantum as tfq
import cirq

# 1. 创建量子比特
qubit = cirq.GridQubit(0, 0)

# 2. 创建参数化量子电路
circuit = cirq.Circuit(
    cirq.X(qubit) ** sympy.Symbol('theta')
)

# 3. 转换为 TFQ 张量
quantum_data = tfq.convert_to_tensor([circuit])

# 4. 创建量子层
quantum_layer = tfq.layers.Expectation()

# 5. 执行量子计算
output = quantum_layer(quantum_data, operators=[cirq.Z(qubit)])
```
