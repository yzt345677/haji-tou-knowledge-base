## 一、量化投资概述

### 什么是量化投资

**定义：** 用数学模型和计算机算法进行投资决策，而非依赖人类主观判断。

**核心特点：**
- 数据驱动 - 基于历史数据和统计规律
- 模型决策 - 用数学模型代替主观判断
- 纪律执行 - 严格按模型信号执行，避免情绪干扰
- 分散投资 - 同时持有大量股票，分散风险

### 量化 vs 主观投资

| 维度 | 量化投资 | 主观投资 |
|------|----------|----------|
| 决策依据 | 数据 + 模型 | 经验 + 判断 |
| 执行方式 | 程序化自动 | 人工手动 |
| 持仓数量 | 几十到几百只 | 几只到十几只 |
| 交易频率 | 高（日/周调仓） | 低（月/季调仓） |
| 情绪影响 | 无 | 大 |
| 适用场景 | 短期交易/套利 | 长期价值投资 |

### Backtrader 核心概念

```python
# 示例：简单移动平均策略
import backtrader as bt

class SmaCross(bt.Strategy):
    params = (('pfast', 10), ('pslow', 30),)
    
    def __init__(self):
        self.sma1 = bt.ind.SMA(period=self.p.pfast)
        self.sma2 = bt.ind.SMA(period=self.p.pslow)
        self.crossover = bt.ind.CrossOver(self.sma1, self.sma2)
    
    def next(self):
        if self.crossover > 0:
            self.buy()  # 金叉买入
        elif self.crossover < 0:
            self.sell()  # 死叉卖出
```

### 回测流程

1. **数据准备** - 获取历史行情数据（OHLCV）
2. **策略定义** - 编写买入/卖出逻辑
3. **回测执行** - 模拟历史交易
4. **性能分析** - 计算收益/风险指标
5. **参数优化** - 调整策略参数
6. **实盘部署** - 接入实盘交易

## 三、核心性能指标

### 收益指标

| 指标 | 公式 | 含义 | 优秀标准 |
|------|------|------|----------|
| 年化收益率 | (期末值/期初值)^(1/年数) - 1 | 平均每年收益率 | >15% |
| 累计收益率 | (期末值 - 期初值) / 期初值 | 总收益率 | >50% |
| 阿尔法（α） | 策略收益 - 基准收益 | 超额收益 | >0 |
| 胜率 | 盈利交易次数 / 总交易次数 | 盈利概率 | >50% |

### 风险指标

| 指标 | 公式 | 含义 | 优秀标准 |
|------|------|------|----------|
| 最大回撤 | (峰值 - 谷值) / 峰值 | 最大亏损幅度 | <20% |
| 波动率 | 收益率标准差 | 价格波动程度 | <30% |
| 贝塔（β） | 策略与基准的协方差/基准方差 | 系统性风险 | ≈1 |
| VaR（风险价值） | 95% 置信度下的最大损失 | 极端风险 | <5% |

### 风险调整收益指标

| 指标 | 公式 | 含义 | 优秀标准 |
|------|------|------|----------|
| 夏普比率 | (年化收益 - 无风险利率) / 波动率 | 单位风险的超额收益 | >1 |
| 索提诺比率 | (年化收益 - 无风险利率) / 下行波动率 | 只考虑下行风险 | >1.5 |
| 卡玛比率 | 年化收益 / 最大回撤 | 收益回撤比 | >2 |
| 信息比率 | 阿尔法 / 跟踪误差 | 超额收益的稳定性 | >0.5 |

## 四、常见量化策略类型

### 1. 趋势跟踪策略

**原理：** 追涨杀跌，顺势而为

**常见方法：**
- 移动平均线交叉（MA Cross）
- 动量策略（Momentum）
- 通道突破（Channel Breakout）
- MACD 指标

**适用场景：** 趋势明显的市场

**风险：** 震荡市频繁止损

---

### 2. 均值回归策略

**原理：** 价格围绕价值波动，高抛低吸

**常见方法：**
- 布林带策略（Bollinger Bands）
- RSI 超买超卖
- 统计套利（Pairs Trading）
- 网格交易

**适用场景：** 震荡市

**风险：** 趋势市逆势亏损

---

### 3. 多因子选股策略

**原理：** 用多个因子筛选股票，构建投资组合

**常见因子：**
- **价值因子：** PE、PB、PS、股息率
- **成长因子：** 营收增速、净利润增速
- **动量因子：** 过去 N 日收益率
- **质量因子：** ROE、毛利率、负债率
- **规模因子：** 市值、流通市值

**构建流程：**
1. 因子筛选（IC 检验）
2. 因子加权（等权/IC 加权/机器学习）
3. 股票排序（综合得分）
4. 组合构建（选前 N 只）
5. 定期调仓（周/月）

**适用场景：** A 股长期有效

---

### 4. 统计套利策略

**原理：** 利用统计规律进行套利

**常见方法：**
- 配对交易（Pairs Trading）
- 期现套利
- 跨市场套利
- ETF 套利

**特点：** 低风险、低收益、高频率

**适用场景：** 市场无效性明显时

## 五、量化交易风险

### 1. 过拟合（Overfitting）

**问题：** 策略在历史数据上表现完美，实盘就失效

**原因：**
- 参数过多，过度优化
- 数据挖掘偏差
- 样本外数据不足

**解决方法：**
- 简化策略，减少参数
- 样本外测试（Out-of-Sample）
- 交叉验证
- 逻辑优先，数据验证

---

### 2. 幸存者偏差（Survivorship Bias）

**问题：** 回测只用当前存在的股票，忽略了退市股票

**影响：** 回测收益虚高

**解决方法：**
- 使用完整历史数据（含退市股票）
- 考虑 IPO 和退市时间点

---

### 3. 前瞻性偏差（Look-ahead Bias）

**问题：** 回测中使用了未来数据

**例子：**
- 用年报数据但在年报披露前交易
- 用收盘价计算信号但当天就交易

**解决方法：**
- 严格使用当时可得数据
- 考虑数据披露时间
- 设置合理的交易延迟

---

### 4. 交易成本忽略

**问题：** 回测不考虑手续费、滑点、冲击成本

**影响：** 高频策略收益虚高

**解决方法：**
- 加入手续费（万 2.5-万 3）
- 加入滑点（0.1%-0.5%）
- 考虑冲击成本（大资金）

---

### 5. 市场结构变化

**问题：** 历史规律在未来可能失效

**原因：**
- 政策变化
- 市场参与者结构变化
- 交易规则变化

**解决方法：**
- 定期更新策略
- 多策略分散
- 设置策略失效预警
