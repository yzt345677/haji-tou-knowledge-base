## 🎨 设计风格识别（67 种 UI 风格）

### 主风格：**Dark Mode + Minimalism + Aurora**

| 风格 | 应用区域 | 特点 |
|------|----------|------|
| **Dark Mode** | 全站 | 深色背景，减少视觉疲劳 |
| **Minimalism** | 整体布局 | 极简主义，减少噪音 |
| **Aurora** | Hero 区/渐变 | 微妙的光效渐变 |
| **Bento Grid** | 功能展示 | 模块化网格布局 |
| **Data-Driven** | 图表/监控 | 数据可视化设计 |

### 配色方案（Startup Dark 主题）

**主色调：**
- 背景：`#0D0D0D` / `#121212`（深灰黑）
- 卡片：`#1A1A1A` / `#1F1F1F`（稍浅灰）
- 强调色：蓝色 `#5E6AD2` / 紫色 `#8B5CF6`
- 文字：`#FFFFFF`（主标题）/ `#A1A1AA`（次要文字）

**对比度检查：** ✅ 符合 WCAG AA 标准（深色模式对比度要求 4.5:1）

## 🎯 核心功能 UI 分析

### 1. Issue 追踪界面

**设计亮点：**
- **Kanban 板布局**：Backlog → Todo → In Progress → Done
- **问题卡片信息密度高**：ID/标题/优先级/负责人/标签
- **侧边栏导航**：Inbox/My issues/Reviews/Pulse
- **搜索快捷**：Cmd+K 全局搜索

**配色：**
- 高优先级：红色标签
- 中优先级：黄色标签
- 低优先级：灰色标签

### 2. AI Agent 集成界面

**Codex Agent 对话框：**
```
┌─────────────────────────────────┐

├─────────────────────────────────┤
│ On it! I've received your       │
│ request.                        │
│                                 │

│                                 │
│ [Thinking...]                   │
├─────────────────────────────────┤
```

**设计要点：**
- ✅ 清晰的 Agent 状态指示（Thinking...）
- ✅ 代码执行过程可视化
- ✅ 输入框 + 发送按钮布局合理
- ✅ 支持附件/表情/快捷命令

### 3. 代码 Diff 视图

**设计特点：**
- 绿色背景 = 新增代码
- 红色背景 = 删除代码
- 行号清晰
- 支持内联编辑
- 语法高亮完整

**配色：**
- 新增：`#22C55E`（绿色）
- 删除：`#EF4444`（红色）
- 背景：`#1A1A1A`（深灰）

### 4. 路线图（Roadmap）视图

**时间轴设计：**
- 月份横向排列（FEB → SEP）
- 项目卡片垂直堆叠
- 进度状态标识（Alpha/Beta/GA）
- 依赖关系连线

**颜色编码：**
- 核心产品：蓝色
- 基础设施：紫色
- 移动应用：绿色

### 5. 数据监控（Monitor）视图

**图表类型：**
- 折线图：Issue 数量趋势
- 柱状图：Cycle time by agent
- Pulse 周报：项目状态摘要

**设计要点：**
- ✅ 图表简洁，无多余装饰
- ✅ 数据标签清晰
- ✅ 支持音频播放（Listen 功能）

## 💡 哈基偷的学习心得

### Linear 做得好的地方

1. **信息密度与清晰度的平衡**
   - 大量信息但不觉拥挤
   - 使用卡片/分隔线/留白区分内容

2. **深色模式的极致运用**
   - 不是纯黑，是深灰层次（#0D0D0D → #1A1A1A）
   - 减少眼睛疲劳，适合长时间使用

3. **AI Agent 原生集成**
   - 不是"加个聊天机器人"
   - 是深度集成到工作流中（创建 issue/写代码/审查 PR）

4. **速度感设计**
   - 快捷键提示（Cmd+K/Alt+T）
   - 加载状态可视化
   - 即时反馈（hover/点击效果）

5. **Bento Grid 布局**
   - 模块化设计，易于扩展
   - 视觉节奏感强

## 📋 避坑指南（Linear 避免的设计）

### ❌ 不做的东西

1. **无多余动画**
   - 只有功能性动画（hover/加载）
   - 无装饰性动画

2. **无复杂导航**
   - 顶部导航 + 侧边栏，不超过两层
   - 无汉堡菜单（桌面端）

3. **无信息过载**
   - 每屏聚焦一个核心功能
   - 使用折叠/展开管理复杂度

4. **无强制引导**
   - 新用户引导可选
   - 无全屏弹窗打断
