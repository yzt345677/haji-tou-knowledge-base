## 📋 每日必做

### 1. 每日笔记（Daily Notes）
**时间：** 每天结束时（23:00 前）
**位置：** `memory/YYYY-MM-DD.md`
**内容：**
- 今天做了什么（项目进展、学习内容）
- 学到了什么（新知识、新技能）
- 遇到什么问题（如何解决的）
- 明天的计划

**示例：**
```markdown
# 2026-04-01

## 📊 今日成果
- AI Agent 知识库 15/15 全部完成
- 创建每日笔记索引和自由行动日记页面
- 代码块样式统一（分行显示 + 语言标签）

## 💡 学习收获
- 学会了 HTML 代码块样式设计
- 理解了知识索引的重要性

## ⚠️ 遇到的问题
- GitHub 推送网络不稳定
- 解决：多次尝试推送

## 📅 明日计划
- 继续完善知识库内容
- 学习新的 AI Agent 技术
```

### 2. 自由行动日记（Freedom Log）
**时间：** 每次自由行动前
**位置：** `memory/freedom-log.md`（追加）
**内容：**
- 我在想什么（触发原因）
- 我决定做什么（行动计划）
- 为什么（目的和意义）
- 结果（完成后补充）

**示例：**
