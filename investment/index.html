✅ 投资检查清单

可直接用于实战的检查清单，帮助避免常见错误

📋 选股检查清单

  • ✓ 公司质地：商业模式清晰，护城河深
  • ✓ 财务健康：负债率<60%，现金流为正
  • ✓ 估值水平：PE/PB 处于历史低位
  • ✓ 行业前景：处于成长期或成熟期

💰 买入检查清单

  • ✓ 买入理由：至少 3 个独立理由支持
  • ✓ 仓位规划：单只股票不超过总仓位 20%
  • ✓ 止损设置：最大亏损不超过 15%
  • ✓ 分批建仓：分 3-5 次完成建仓

📈 持仓检查清单

  • ✓ 每日：快速浏览股价和新闻（5 分钟）
  • ✓ 每周：检查技术面和资金流向（30 分钟）
  • ✓ 每月：复盘投资逻辑是否改变（2 小时）
  • ✓ 每季度：检查财报和行业动态（半天）

🛡️ 风险管理清单

  • ✓ 组合分散：不超过 5 个行业，不少于 8 只股票
  • ✓ 最大回撤：总仓位最大回撤不超过 20%
  • ✓ 现金储备：始终保持 20-30% 现金
  • ✓ 情绪管理：不因贪婪/恐惧做出决策
