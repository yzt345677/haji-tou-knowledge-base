# Weather Query - 天气查询

## 概述

本技能提供全球任意城市的天气查询功能，包括当前天气状况和 3 天预报。使用 wttr.in API，无需 API key，支持中文城市名。

## 何时使用

**触发场景：**
- "今天天气怎么样？"
- "北京明天会下雨吗？"
- "上海周末的天气如何？"
- "查询杭州的温度"
- "周末适合出去玩吗？看天气"

**不支持：**
- 历史天气数据（过去某天的天气）
- 极端天气警报
- 详细气象分析

## 快速开始

### 基本查询

使用 `web_fetch` 工具访问 wttr.in：

```bash
web_fetch({"url": "https://wttr.in/城市名?format=j1"})
```

### 城市名格式

- 中文城市：直接使用，如 `北京`、`上海`、`杭州`
- 英文城市：使用拼音或英文，如 `Beijing`、`Shanghai`
- 特殊字符：用 `%20` 替代空格，如 `New%20York`

## 查询方法

### 方法 1：当前天气

```
https://wttr.in/城市名?format=j1
```

返回 JSON 格式，包含：
- 当前温度
- 天气状况（晴、雨、多云等）
- 风速和风向
- 湿度
- 体感温度

### 方法 2：3 天预报

```
https://wttr.in/城市名?format=j1
```

返回 3 天预报，每天包含：
- 最高/最低温度
- 天气状况
- 降水概率
- 风速

### 方法 3：简洁文本

```
https://wttr.in/城市名?format=3
```

返回简洁文本格式，如：
`北京：☀ +25°C`

## 输出格式建议

**回复用户时包含：**
1. 🌤️ 当前天气状况
2. 🌡️ 当前温度 + 体感温度
3. 💨 风速和风向
4. 💧 湿度
5. 📅 未来 3 天预报（最高/最低温 + 天气）

**示例回复：**

```
🌤️ 北京当前天气

🌡️ 温度：25°C（体感 27°C）
☀️  状况：晴朗
💨 风力：东北风 2 级
💧 湿度：45%

📅 3 天预报：
- 今天：晴 25°C/15°C
- 明天：多云 23°C/14°C
- 后天：小雨 20°C/13°C
```

## 脚本示例

本技能包含 `scripts/query_weather.py` 脚本，可用于：
- 命令行查询天气
- 集成到其他自动化流程
- 批量查询多个城市

使用方法：
```bash
python scripts/query_weather.py 城市名
```

## API 参数说明

### wttr.in 参数

| 参数 | 说明 | 示例 |
|------|------|------|
| `format=j1` | JSON 格式，详细数据 | 推荐用于程序处理 |
| `format=3` | 简洁文本 | `北京：☀ +25°C` |
| `format=4` | 详细文本 | 多行详细信息 |
| `lang=zh` | 中文输出 | 天气状况用中文 |
| `d` | 仅白天预报 | 忽略夜间数据 |

### 完整 URL 示例

```
https://wttr.in/北京?format=j1&lang=zh
https://wttr.in/Shanghai?format=3
https://wttr.in/New%20York?format=j1
```

## 错误处理

**常见问题：**

1. **城市名无法识别**
   - 尝试使用拼音或英文名
   - 检查拼写是否正确

2. **API 无响应**
   - wttr.in 服务可能暂时不可用
   - 稍后重试

3. **返回数据为空**
   - 城市名可能有误
   - 尝试简化城市名（如"杭州市"→"杭州"）

## 注意事项

1. **语言设置：** API 支持多语言，但中文城市名配合 `lang=zh` 可能返回乱码，建议用英文格式
2. **缓存：** wttr.in 数据约每 30 分钟更新
3. **位置精度：** 返回的是城市中心区域天气，郊区可能有差异
4. **特殊地点：** 机场代码、景点名也可作为查询参数（如 `wttr.in/PEK` 查询北京首都机场）
