✅ 完整检查清单

上线前必查项目，包含基础/无障碍/性能/响应式/SEO 五大类

📋

基础检查

  • 颜色一致
  • 字体统一
  • 间距规律
  • 无错别字
  • 链接有效
  • 表单可用
♿

无障碍检查 (WCAG 2.1 AA)

  • 文本对比度 ≥ 4.5:1
  • 焦点状态可见
  • 可键盘访问
  • 图片 alt 文本
  • 表单 label
⚡

性能检查 (Core Web Vitals)

  • LCP < 2.5s
  • FID < 100ms
  • CLS < 0.1
  • 图片优化
  • 代码压缩
📱

响应式检查

  • 手机小 (375px)
  • 手机大 (414px)
  • 平板 (768px)
  • 桌面小 (1024px)
  • 桌面大 (1440px)
